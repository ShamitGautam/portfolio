# Tech Profile Hub

This is my personal tech portfolio project where I showcase my skills, projects, and development work.

## About the Project

Tech Profile Hub is a modern web application built to display my technical profile, projects, and experience in a clean and responsive interface.

## Technologies Used

* React
* TypeScript
* Vite
* Tailwind CSS
* shadcn/ui

## How to Run Locally

```bash
git clone https://github.com/ShamitGautam/portfolio.git
cd tech-profile-hub
npm install
npm run dev
```

## Features

* Modern UI Design
* Responsive Layout
* Fast Development using Vite
* Clean Component Structure
